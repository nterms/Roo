Getting Started

Please note that Roo is still in beta release and may contain many bugs and incomplete functions. However this simple tutorial would support you getting started on developing your amazing HTML5 canvas application with Roo.

Keep in touch, there's more to come!

Download & Install

Roo can be installed in two easy steps. Download and copy it into your scripts folder. In this case we call it js. Then add the following code into your HTML page.

	
<script type="text/javascript" src="js/roo.js" ></script>

Build your Stage

Stage provides the background environment for your graphics application. All the layers of an application should be added to the stage to make them parts of the scene rendered ultimately.

window.onload = function() {
	var stage = new Roo.Stage('container', 600, 400);
}

Where container is the container element for the stage to be added into. Basically this could be a <div> element.

Shapes

You may create a shape object of an existing primitive shape or define your own shape with basic drawing instructions. Creating primitive shape is a one step process. Lets start by creating a Rectangle shape. 

window.onload = function() {
	var stage = new Roo.Stage('container', 600, 400);
	var rect = new Roo.Rectangle(100, 50, 80, 90);
}

A new Rectangle object - rect - is created with the configurations left = 100, top = 50, width = 80 and height = 90.

Shapes section contains more about primitive and custom shapes.

A layered architecture

Objects are added to layers which are subsequently added to the Stage for rendering. See how we can create a layer and add objects to that layer.

window.onload = function() {
	var stage = new Roo.Stage('container', 600, 400);
	var rect = new Roo.Rectangle(100, 50, 80, 90);
	var layer = new Roo.Layer();
	layer.add(rect);
	stage.add(layer);
	stage.update();
}

Note that the layer is added to the stage making it available for rendering and managing events on shape objects of that particular layer. Update the stage to render its content and make them visible.

Events

You may attach several events to the objects in your scene. To get you started on event handling, lets add a click event to the rectangle we created.

window.onload = function() {
	var stage = new Roo.Stage('container', 600, 400);
	var rect = new Roo.Rectangle(100, 50, 80, 90);
	var layer = new Roo.Layer();
	layer.add(rect);
	stage.add(layer);
	rect.onclick = function(evt) {
		alert("You clicked on the rectangle!");
	};
	rect.draggable = true;
	stage.update();
}

You may make the objects draggable by setting its draggable property to true.

Putting it all together

	<head>
		<title>Roo - Sample Application</title>
		<script type="text/javascript" src="js/roo.js" ></script>

			window.onload = function() {
				var stage = new Roo.Stage('container', 600, 400);
				var rect = new Roo.Rectangle(100, 50, 80, 90);
				var layer = new Roo.Layer();
				layer.add(rect);
				stage.add(layer);
				rect.onclick = function(evt) {
					alert("You clicked on the rectangle!");
				};
				rect.draggable = true;
				stage.update();
			}
		</script>
	</head>
	<body>
		<div id="container"></div>
	</body>
</html>

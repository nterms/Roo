Class Detail
Stage(container, width, height)
The Stage class provides the basement for all the graphics layers of an application to be placesd on. Stage is the root container for all the graphics objects created with Roo. It works as the primary event manager of an application as well.
Parameters:
container
ID of the container element of the Stage.
width
Width of the stage in pixels
height
Height of the stage in pixels

{Stage} enableEvents(events)
Enable event listening on the stage.
All supported events:
  • click
  • mousedown
  • mouseup
  • mousemove
  • mouseover
  • mouseout
  • touchstart
  • touchend
  • touchmove
This method is used to enable all supported events or a selected set of them. Enabling only the reqired events is reccomended as event management is an expensive process consumes more processing power. 

stage.enableEvents();

stage.enableEvents(['click']);

stage.enableEvents(['click', 'mousemove', 'mousedown']);
Parameters:
{Object} events
An array of event names to be enabled. Omitting this argument would enable all the supported events.
Returns:
{Stage} The Stage object
